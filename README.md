# The Hackropad
A small 3-key macropad called a hackropad that will help me code faster by typing common Java snippets like if-else statements, for loops, and debug prints with a single key press.

## Keymap

* Key 1 → if / else block
* Key 2 → for loop
* Key 3 → debug print

## How to use

1. Plug it in via USB
2. It acts like a keyboard (no setup needed)
3. Press a key to insert the snippet in your editor

## Firmware

KMK (CircuitPython)

Main file:

```text
Firmware/main.py
```

## Hardware

Direct pin layout:

* D10
* D9
* D8
